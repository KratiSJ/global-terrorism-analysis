# Global Terrorism Data Analysis Dashboard

## 📌 Project Overview
The Global Terrorism Database (GTD) is an open-source database that contains information about terrorist attacks around the world from 1970 to 2017. The dataset includes both domestic and international terrorist incidents and contains over 180,000 recorded attacks.

This project focuses on analyzing global terrorism trends using Power BI to uncover meaningful insights related to attack types, target types, weapons used, regions affected, and terrorist organizations.

The dashboard was created using raw Excel data and transformed into an interactive business intelligence report for better understanding and visualization of terrorism patterns worldwide.

---

## 🎯 Project Objectives
- Analyze terrorism incidents across different countries and regions
- Identify the most successful attack types and weapon types
- Discover the most affected target categories
- Track yearly attack trends
- Identify the most deadly terrorist groups
- Build an interactive dashboard for dynamic analysis

## 📊 Dashboard Features

### Visualizations Included

#### 1️⃣ Success Rate by Type of Attack
Analyzes successful terrorist incidents based on attack types such as:
- Bombing/Explosion
- Armed Assault
- Assassination

#### 2️⃣ Success Rate by Target Type
Shows the most targeted sectors including:
- Private Citizens & Property
- Military
- Police

#### 3️⃣ Success Rate by Weapon Type
Displays the success rate based on weapon categories:
- Explosives
- Firearms
- Unknown Weapons

#### 4️⃣ Success Rate by Country
Interactive world map displaying country-wise terrorism activity.

#### 5️⃣ Success Rate by Region
Compares terrorism incidents across global regions.

#### 6️⃣ Success Rate by Name of Group
Highlights the most active terrorist organizations.

#### 7️⃣ Total Attacks by Year
Shows yearly terrorism trends from 1970 to 2017.

---

## 📈 Key Insights
- Bombing/Explosion is the most common attack type.
- Explosives are the most frequently used weapons.
- Middle East & North Africa recorded the highest terrorism activity.
- Taliban emerged as one of the most deadly terrorist organizations.
- Terrorist incidents increased significantly after the early 2000s.
